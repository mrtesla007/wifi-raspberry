#### Installing dependencies

If you are using Yarn, run: 

```bash
yarn global add webpack-cli
yarn
```

in the root of the project. 

If you are using ``npm``, run 

```bash
sudo npm install -g webpack-cli
npm install
```

in the root of the project.
